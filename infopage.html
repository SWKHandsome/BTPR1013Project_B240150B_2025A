Personal Protection

Use Strong Passwords

    All passwords should be:
  • Long: At least 16 characters
  • Unique: Never reuse passowrds
  • Random: Use a random string of mixed-case letters,
    numbers and symbols, like: Yuc8^RikA34%ZoPPao98t

Avoid using easily guessable information like birthdays or common words and don't tell anyone your passwords!

Turn on Multi-factor Authentication

  • Verify their identity
  • Creates an extra hurdle for password thieves

Secure Internet Connections

  • Avoid using public Wi-Fi networks
  • Use a virtual private network (VPN)

Watch Out for Internet Scams

  • Don't download attachments if they're not from a known and trusted contact.
  • Be skeptical of emails or text messages

Data Security

Improve Network Security

  • Using firewalls to block unauthorized access.
  • implementing intrusion detecftion systems to monitor for and prevent cyber-attacks

Implement Access Controls

  • Reducing the risk of data breaches and unauthorized access.
  • Restricting access to sensitive information to only authorized users.

Improve Physical Security

  • Secure physical devices and facilities that store sensitive information

Encrypt Your Sensitive Data

  • Converting sensitive information into a coded form.
  • Storing or sharing sensitive data to preserve its confidentiality.
